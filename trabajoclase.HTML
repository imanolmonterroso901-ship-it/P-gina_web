¿Qué es HTML5?

HTML5 es la última versión del lenguaje de marcado HTML que introduce nuevas etiquetas semánticas, soporte multimedia, y más.

Nuevas Etiquetas Semánticas

Etiquetas como <section>, <article> y <aside> ayudan a estructurar mejor el contenido de una página web.

Sabías que...

La etiqueta <aside> se usa para mostrar contenido relacionado pero no principal.
